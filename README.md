# Прогноз вероятности покупки (90 дней)

## Структура
- `notebooks/` — основной анализ и моделирование.
- `data/` — (в игноре) исходные данные.

## Описание проекта
Цель проекта — предсказать вероятность совершения покупки клиентом в течение следующих 90 дней. Это позволяет маркетингу более эффективно распределять бюджет на рассылки, фокусируясь на «горячих» пользователях или реанимируя тех, кто склонен к покупке, но колеблется.

**Целевая метрика:** ROC-AUC

## Данные
- `apparel-purchases`: История покупок (id клиента, цена, категории, количество).
- `apparel-messages`: История маркетинговых взаимодействий (отправки, открытия, клики).
- `apparel-target_binary`: Целевой признак (факт покупки в течение 90 дней).

## Используемый стек
- **Язык**: Python 3.10+
- **Библиотеки**: Pandas, Scikit-learn, CatBoost, Matplotlib, Seaborn.
- **Инструменты**: VS Code, Jupyter Notebook.

## Основные этапы работы
1. **EDA**: Анализ дисбаланса классов (целевой класс — 1.9%) и очистка данных от дубликатов.
2. **Feature Engineering**:
   - Построение RFM-метрик (Recency, Frequency, Monetary).
   - Агрегация маркетинговой активности (Open Rate, частота взаимодействий).
   - *В разработке*: Обработка иерархических категорий товаров.
3. **Моделирование**: Сравнение Random Forest и CatBoost.
4. **Оптимизация**: Применение весов классов для борьбы с дисбалансом.

## Как запустить
1. Клонировать репозиторий.
2. Установить зависимости: `pip install -r requirements.txt`.
3. Положить данные в папку `data/raw/`.
4. Запустить ноутбук `notebooks/marketing_purchase_prediction.ipynb`.

## Текущие результаты
- **Random Forest (Baseline)**: ROC-AUC ~0.69
- **CatBoost**: В процессе настройки.

## Итоговые результаты
- **Baseline (Random Forest)**: ROC-AUC 0.6911
- **Final Model (CatBoost)**: ROC-AUC 0.7514

## Ключевые выводы
- Самый важный признак — `recency` (давность покупки).
- Маркетинговые метрики (клики и открытия) входят в топ-3 факторов.
- Модель позволяет выделить узкий сегмент клиентов (около 2%), которые принесут основную прибыль в следующие 90 дней.

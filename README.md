# Прогноз вероятности покупки (90 дней)

## Описание
Проект по классификации пользователей интернет-магазина. Цель — выявить клиентов, склонных к покупке в течение 90 дней, для оптимизации маркетинговых кампаний.

## Стек
- Python (Pandas, Scikit-learn, CatBoost/LightGBM)
- Jupyter Notebook
- Matplotlib/Seaborn

## Структура
- `notebooks/` — основной анализ и моделирование.
- `data/` — (в игноре) исходные данные.
